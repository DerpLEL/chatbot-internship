HRM system
API DOCUMENTATION
Endpoints:
- /api/User/me?email=<email>: accepts an email as input. Use GET with this endpoint along with a user's email to get data of a specific user.
- /api/LeaveApplication/<userId>: accepts an ID string as a path variable. Use GET with this endpoint along with a user's ID to get all leave applications submitted by a user.
